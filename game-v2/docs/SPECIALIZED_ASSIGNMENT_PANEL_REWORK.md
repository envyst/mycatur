# Specialized Assignment Panel Rework

## Changes
- white/black are now shown as side buttons/tabs at the top instead of rendering both panels at once
- specialized assignment rows are dynamic instead of fixed to 6
- starts with 1 row by default
- `+ Add row` grows the panel gradually
- specialized piece selector is now a case-insensitive searchable combobox-like field instead of a separate stacked search bar

## Goal
Keep the existing working assignment flow while making the setup panel cleaner and more scalable.
