# Mycatur Replay UX Progress

## Added in this pass
- move log entries are now clickable replay targets
- clicking a move jumps the board to that historical position
- replay navigation remains available with step buttons

## Practical result
Finished-session history is now more usable because the user can jump directly to a move of interest instead of stepping one-by-one every time.

## Still to improve later
- clearer visual difference between replay state and live end position
- optional auto-scroll/highlight behavior for selected move entries
- richer replay metadata per move
