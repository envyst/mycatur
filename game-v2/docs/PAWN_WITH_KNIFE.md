# Pawn with Knife

## Source rule
Pawn with Knife (pawn) = Has +1 diagonal capture range towards the center of the board.

## Current implementation
- normal pawn movement unchanged
- normal pawn captures unchanged
- gains an additional capture square two squares diagonally forward toward the center of the board

## Center direction
- files a-d -> inward direction is right
- files e-h -> inward direction is left
