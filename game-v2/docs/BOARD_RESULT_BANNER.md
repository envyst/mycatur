# Mycatur Board Result Banner

## Added in this pass
A clearer board-area state banner was added so the result is not hidden only in the sidebar status.

### Current board banner behavior
- `CHECKMATE` banner shown near the board with winner text
- `DRAW` banner shown near the board for draw end states
- `REPLAY MODE` banner shown when viewing historical positions instead of the live board

## Why
This makes important game states much more visible during play and when reviewing finished sessions.
