# Specialized Assignment Rules

## Current phase
Assignment only. Effects disabled.

## Per-side rules
- maximum 6 specialized assignments per side
- one specialization per piece
- no duplicate square assignment
- assignments only before the game starts

## Piece matching rule
A specialization can only be assigned to its matching base piece family:
- bishop specializations -> bishops
- knight specializations -> knights
- pawn specializations -> pawns
- rook specializations -> rooks
- queen specializations -> queens
- king specializations -> kings

## Current gameplay rule
Even after assignment, actual gameplay still follows normal chess rules for now.
Specialized identity currently affects setup/persistence/display only.
