# Specialized Visibility and Board Coordinates

## Added in this pass
- board coordinates are now shown directly on the board
- assigned specialized pieces display a visible marker badge
- piece tooltip/title becomes more informative for assigned specialized pieces
- assignment dropdown labels for duplicated piece starts are friendlier, e.g.:
  - `b1 - left knight`
  - `g1 - right knight`

## Why
This makes specialized assignment trustworthy and easier to understand without relying on memory alone.
