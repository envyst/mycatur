# Specialized Mode Phase 1 Plan

## Goal
Implement specialized-piece assignment/setup before implementing any specialized effects.

## Scope
- separate `specialized` mode
- pre-game assignment UI
- max 6 specialized pieces per side
- persist assignments in session/app state
- show specialized identities in UI
- keep gameplay rules identical to normal chess for now

## Explicitly out of scope
- movement changes
- passive effects
- triggered effects
- alternate win conditions
- custom AI behavior
